The goal of this project aims to be ActiveX component like Microsoft Silver Light or server-client application like Autodesk Map guide , Read every model with its attributes (IDs, descriptions, phase…etc), Object selection and browsing, Ability to connect model with databases like oracle or SQL servers through unified IDs ,Adding security layer to prevent/allow users from seeing data. Have good performance/reliability over Internet/intranet.

Features

ActiveX control that can be added into any container in the Microsoft Windows platform.

Uses the famous WCF web service to provide a middleware layer.

Uses SOAP for reliability and security of data exchange.

High performance in rendering and retrieving the model.
